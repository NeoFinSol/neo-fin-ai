# NeoFin AI Frontend Documentation

## Обзор системы
**NeoFin AI** — это современная фронтенд-платформа для автоматизированного финансового анализа PDF-отчетов с использованием искусственного интеллекта. Интерфейс спроектирован с упором на профессиональную аналитику, высокую читаемость данных и real-time взаимодействие.

---

## Основные функции и страницы

### 1. Dashboard (Главная панель)
Центральный хаб для загрузки и первичного анализа документов.
*   **Интеллектуальная загрузка**: Поддержка Drag-and-Drop через Mantine Dropzone.
*   **Real-time Прогресс**: Анимированный индикатор выполнения (Progress Bar), имитирующий этапы загрузки (0-30%) и глубокого анализа ИИ (30-90%).
*   **Мгновенный скоринг**: Отображение ключевых метрик (Ликвидность, Рентабельность, Долг) сразу после завершения обработки.
*   **Система уведомлений**: Интерактивные Toast-уведомления, информирующие о статусе (загрузка, успех, ошибка).

### 2. Detailed Report (Детальный отчет)
Глубокая визуализация результатов анализа.
*   **AI-Инсайты**: Блок текстовой аналитики, оформленный в стиле премиальной бизнес-периодики (шрифт Libre Baskerville, выравнивание по ширине, увеличенный интервал 1.8).
*   **Факторы риска**: Наглядный список факторов с цветовой кодировкой и иконками (TrendingUp/Down/Minus) в зависимости от влияния на рейтинг.
*   **Финансовые коэффициенты**: Сетка карточек с трендами и расшифровкой показателей.
*   **Таблица показателей**: Детальный разбор выручки, прибыли и активов с привязкой к нормативам.
*   **Экспорт**: Функционал печати и подготовки к скачиванию PDF-версии отчета.

### 3. Analysis History (История анализов)
Архив всех проведенных проверок.
*   **Статусная система**: Цветовые бейджи (Готово, Обработка, Ошибка).
*   **Быстрый просмотр**: Отображение итогового балла (Score) с цветовой индикацией уровня риска.
*   **Управление**: Возможность скачивания или удаления записей из архива.

### 4. Settings (Настройки)
Персонализация и управление доступом.
*   **Профиль**: Редактирование данных пользователя и компании.
*   **Безопасность**: Смена пароля и управление 2FA.
*   **API Management**: Генерация и копирование ключей для интеграции с внешними системами.
*   **Биллинг**: Мониторинг лимитов анализов и управление подпиской.

## Технологический стек

*   **Framework**: React 19 + TypeScript.
*   **UI Library**: Mantine v8 (Core, Hooks, Notifications, Dropzone).
*   **Styling**: Tailwind CSS 4.0 (для кастомных раскладок) + Mantine Emotion (для компонентов).
*   **Icons**: Lucide React.
*   **Routing**: React Router 7.
*   **API Client**: Axios с интерцепторами для автоматической передачи API-ключей и обработки 401 ошибок.
*   **Animations**: Motion (framer-motion) для плавных переходов.

---

## Архитектурные особенности

1.  **usePdfAnalysis Hook**: Кастомный хук, инкапсулирующий логику загрузки файла и последующего опроса (polling) бэкенда для получения результата. Включает встроенную обработку таймаутов и сетевых ошибок.
2.  **Auth Context**: Глобальное состояние авторизации, синхронизированное с `localStorage` и событиями хранилища браузера.
3.  **Protected Routes**: Система защиты маршрутов, предотвращающая доступ неавторизованных пользователей к аналитическим данным.
4.  **Error Handling**: Централизованная обработка ошибок API с выводом понятных пользователю уведомлений.

## Интеграция с Backend
Фронтенд ожидает API по адресу `/api` (настраивается через прокси в `vite.config.ts`).
*   `POST /upload`: Загрузка файла, возвращает `task_id`.
*   `GET /result/:taskId`: Опрос статуса и получение финальных данных анализа.
*   **Auth**: Ожидает заголовок `X-API-Key` для всех защищенных запросов.
