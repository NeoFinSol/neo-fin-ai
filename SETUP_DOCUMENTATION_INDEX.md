## 🎬 СЦЕНАРИИ ИСПОЛЬЗОВАНИЯ

### Сценарий 1: Первый раз открываю проект

```
1. Откройте START_HERE.md
2. Выполните .\env\Scripts\Activate.ps1
3. Выполните python src/app.py
4. Откройте http://localhost:8000/docs
5. ✅ Готово!
```

### Сценарий 2: Нужно запустить тесты

```
1. Активируйте окружение
2. python -m pytest tests/ -v
3. Смотрите результаты
4. ✅ Всё прошло успешно!
```

### Сценарий 3: Нужно разработать новую функцию

```
1. Запустите тесты: python -m pytest tests/ -v
2. Напишите код в src/
3. Запустите тесты: python -m pytest tests/test_*.py -v
4. Форматируйте: black src/
5. Проверяйте: flake8 src/
6. ✅ Git commit и push!
```

### Сценарий 4: Что-то не работает

```
1. Проверьте версию Python: python --version
2. Проверьте что окружение активировано: (env) в терминале?
3. Прочитайте раздел "Решение проблем" в нужном файле
4. Если ошибка про зависимости:
   - PDFMINER_FIX.md
   - PYTHON_VERSION_FIX.md
5. Если другое:
   - Посмотрите QUICK_START.md (раздел FAQ)
```

## ✅ ШПАРГАЛКА КОМАНД

```powershell
# Активировать окружение
.\env\Scripts\Activate.ps1

# Запустить приложение
python src/app.py

# Запустить все тесты
python -m pytest tests/ -v

# Запустить конкретный тест
python -m pytest tests/test_auth.py -v

# Запустить с покрытием
python -m pytest tests/ --cov=src --cov-report=html

# Форматировать код
black src/ tests/

# Проверить стиль
flake8 src/ tests/

# Проверить типы
mypy src/

# Обновить зависимости
python -m pip install -r requirements.txt --upgrade
```
