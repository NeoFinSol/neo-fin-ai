## 💡 Ключевые моменты

1. **Python НЕ требует компиляции**
   - "Build" = установка зависимостей
   - Не нужно нажимать "Build Solution"

2. **Используйте виртуальное окружение**
   - Все пакеты изолированы в папке `env/`
   - Не нужно ничего устанавливать глобально

3. **Всегда активируйте окружение**
   - `.\env\Scripts\Activate.ps1` перед работой
   - В начале строки появится `(env)`

4. **VS Code лучше чем VS для Python**
   - Намного быстрее
   - Лучше поддержка Python
   - Бесплатно и легче
