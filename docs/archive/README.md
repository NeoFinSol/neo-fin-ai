# Архив документации

Каталог `docs/archive/` хранит материалы, которые **намеренно остаются в публичном репозитории**, но **не входят в основное продуктовое ядро** навигации.

## `contest/`

Исторический конкурсный контур (runbook, операторская карта, генератор презентации). Это не эксплуатационная документация текущего продукта и не заменяет `docs/ARCHITECTURE.md`, `docs/API.md` и связанные файлы.

Перенос сюда **не делает** содержимое приватным: всё, что лежит в открытом репозитории на GitHub, остаётся публичным.
