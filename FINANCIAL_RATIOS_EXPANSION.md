## 📐 Формулы коэффициентов

### ГРУППА 1: Коэффициенты ликвидности

#### 1. Коэффициент текущей ликвидности
```
Текущая ликвидность = Текущие активы / Краткосрочные обязательства
Current Ratio = Current Assets / Short-term Liabilities
```
**Интерпретация:** > 1.5 - хорошо, 1.0-1.5 - норма, < 1.0 - риск

#### 2. Коэффициент быстрой ликвидности (НОВЫЙ)
```
Быстрая ликвидность = (Текущие активы - Запасы) / Краткосрочные обязательства
Quick Ratio = (Current Assets - Inventory) / Short-term Liabilities
```
**Интерпретация:** > 1.0 - хорошо, более консервативная метрика

#### 3. Коэффициент абсолютной ликвидности (НОВЫЙ)
```
Абсолютная ликвидность = Денежные средства / Краткосрочные обязательства
Absolute Ratio = Cash & Equivalents / Short-term Liabilities
```
**Интерпретация:** > 0.2 - безопасно, показывает немедленную платежеспособность

---

### ГРУППА 2: Коэффициенты рентабельности

#### 4. ROA (Return on Assets)
```
ROA = Чистая прибыль / Активы всего
ROA = Net Profit / Total Assets
```
**Интерпретация:** > 10% - отличный результат

#### 5. ROE (Return on Equity)
```
ROE = Чистая прибыль / Собственный капитал
ROE = Net Profit / Equity
```
**Интерпретация:** > 15% - хороший результат

#### 6. ROS - Рентабельность продаж (НОВЫЙ)
```
ROS = Чистая прибыль / Выручка
ROS = Net Profit / Revenue
```
**Интерпретация:** > 10% - хорошо, показывает прибыльность продаж

#### 7. EBITDA маржа (НОВЫЙ)
```
EBITDA маржа = EBITDA / Выручка
EBITDA Margin = EBITDA / Revenue
```
**Интерпретация:** > 20% - отличный результат, показывает операционную эффективность

---

### ГРУППА 3: Коэффициенты финансовой устойчивости

#### 8. Коэффициент автономии
```
Автономия = Собственный капитал / Активы всего
Autonomy = Equity / Total Assets
```
**Интерпретация:** > 0.5 (50%) - хорошо, более 0.7 - отлично

#### 9. Финансовый рычаг (НОВЫЙ)
```
Финансовый рычаг = Обязательства / Собственный капитал
Financial Leverage = Liabilities / Equity
```
**Интерпретация:** < 1.0 - безопасно, < 2.0 - приемлемо

#### 10. Покрытие процентов (НОВЫЙ)
```
Покрытие процентов = EBIT / Процентные расходы
Interest Coverage = EBIT / Interest Expense
```
**Интерпретация:** > 5 - очень безопасно, > 2 - приемлемо, < 1 - риск неплатежа

---

### ГРУППА 4: Коэффициенты деловой активности

#### 11. Оборачиваемость активов (НОВЫЙ)
```
Оборачиваемость активов = Выручка / Активы всего
Asset Turnover = Revenue / Total Assets
```
**Интерпретация:** > 1.5 - хорошо, показывает эффективность использования активов

#### 12. Оборачиваемость запасов (НОВЫЙ)
```
Оборачиваемость запасов = Себестоимость / Средние запасы
Inventory Turnover = COGS / Average Inventory
```
**Интерпретация:** > 5 - хорошо, показывает скорость реализации товаров

#### 13. Оборачиваемость дебиторской задолженности (НОВЫЙ)
```
Оборачиваемость ДЗ = Выручка / Дебиторская задолженность
Receivables Turnover = Revenue / Accounts Receivable
```
**Интерпретация:** > 10 - хорошо, показывает скорость сбора платежей
